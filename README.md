# OrzBLE

蓝牙模块相关应用单独成库

# 支持控制的设备列表

- [x] 小米床头灯XMCTD01YL

# 使用

- iOS 10及以上的系统中，App需要在自己的Info.plist中设置`NSBluetoothPeripheralUsageDescription`，，否则App会Crash

# 蓝牙BLE相关的概念

- 中心(Central)
- 外设(Peripheral)
